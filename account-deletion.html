계정 삭제 요청 방법

아래 이메일로 계정 삭제를 요청해 주세요:

support@rezam.kr

이메일 제목: [RE:Zam] 계정 삭제 요청

이메일 내용: 가입 시 사용한 이메일 주소 또는 닉네임을 기재해 주세요.

삭제 처리 절차

  1. 삭제 요청 접수: 위 이메일로 삭제 요청을 보내주세요.
  2. 본인 확인: 요청자 본인 확인을 위해 가입 정보를 확인합니다.
  3. 삭제 처리: 요청 접수 후 7일 이내에 계정 및 데이터가 삭제됩니다.
  4. 완료 통지: 삭제 완료 후 이메일로 안내드립니다.

삭제되는 데이터

계정 삭제 시 다음 데이터가 영구적으로 삭제됩니다:

데이터 유형 삭제 여부
계정 정보 (이메일, 닉네임, 성별, 출생연도) 즉시 삭제
프로필 정보 (프로필 이미지, 목표 등) 즉시 삭제
캘린더 기록 (일일 평가, 메모, 약 복용 기록) 즉시 삭제
다짐 기록 즉시 삭제
라운지 게시글 및 댓글 즉시 삭제
앱 사용 기록 즉시 삭제

주의사항

  • 계정 삭제 후에는 데이터를 복구할 수 없습니다.
  • 삭제된 계정으로는 다시 로그인할 수 없습니다.
  • 동일한 이메일로 재가입은 가능하지만, 이전 데이터는 복구되지 않습니다.

데이터 보관

원칙적으로 모든 데이터는 즉시 삭제됩니다. 다만, 관련 법령에 따라 일부 정보는 다음 기간 동안 보관될 수 있습니다:

  • 전자상거래 등에서의 소비자 보호에 관한 법률에 따른 기록: 최대 5년
  • 통신비밀보호법에 따른 로그 기록: 3개월
